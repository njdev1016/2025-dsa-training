## Intensive Learning Path (3 Weeks, 5 hours/day)

- **Hour 5:** Practice 10-12 sorting problems

#### Day 6: Challenge Day - Apply Week 1 Knowledge (5 hours)

## Key Success Tips for Intensive Learning:

1. **Consistency is crucial** - Don't skip days, momentum is everything
2. **Quality over quantity** - Understand concepts deeply, don't rush
3. **Active learning** - Code along, don't just read
4. **Immediate practice** - Apply concepts within 30 minutes of learning
5. **Test yourself** - Close the tutorial and implement from memory
6. **Rest properly** - 8 hours sleep, good nutrition, exercise
7. **Track progress** - Use a journal to note daily achievements
8. **Review regularly** - Spend 30 min daily reviewing previous days
9. **Ask for help** - Use online communities when stuck (max 30 min)
10. **Celebrate wins** - Reward yourself after completing each challenge

## Challenge Details Reference:

### Part 01 - Wrong Return (Day 6)
Find the 4 most common digits in a sequence. If tied, higher digit comes first.
**Topics:** Frequency counting, Hash maps, Sorting

### Part 02 - White Noise (Day 5)
Calculate total rotation in degrees (0-359) from a sequence of rotations.
**Topics:** Modulo arithmetic, Array traversal

### Part 03 - Game Begins (Day 2)
Find unique letters in text.
**Topics:** String manipulation, Hash sets

### Part 04 - Kitchen Secret (Day 10)
Navigate a 4x4 keypad grid following UP/DOWN/LEFT/RIGHT directions.
**Topics:** 2D grid navigation, Boundary checking

### Part 05 - Piano Puzzle (Day 14)
Find shortest substring containing all 7 notes (A-G).
**Topics:** Sliding window, Hash map

### Part 06 - Runtime Warp (Day 6)
Calculate total Euclidean distance between 4D vectors, rounding up non-integers.
**Topics:** Mathematical operations, Distance formulas

### Part 07 - Phone Call (Day 6)
Convert decimal to binary with 32-bit overflow handling.
**Topics:** Number systems, Binary conversion, Bit manipulation

### Part 08 - Eldritch Horror (Day 6)
Find the word that has no anagram in the dataset.
**Topics:** Anagram detection, Hash maps, String sorting

### Part 09 - You Cannot Escape (Day 19)
Find shortest path in a grid from P to G avoiding walls (#).
**Topics:** BFS, Graph traversal, Shortest path

### Part 10 - Good Ending (Day 4)
Find 2 unpaired letters (one from left 8, one from right 8) in a 16-letter string.
**Topics:** Character counting, Hash maps
